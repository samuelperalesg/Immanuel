# Immanuel

A web application that helps users log their bible verses and give them context with a description and/or links to videos, articles, or pictures.

## Technologies Used
* HTML5
* CSS3
* JavaScript
* jQuery
* Google Fonts
* MongoDB
* Express
* Node
